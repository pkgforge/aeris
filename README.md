# aeris

**Unbounded package management** - A universal GUI for managing packages across all your package managers.

## Overview

Aeris is a graphical interface that unifies package management across different systems. Built with Rust and [iced](https://iced.rs), it provides a clean, modern way to search, install, update, and remove packages from multiple sources through a single interface.
